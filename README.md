# Tema 9 · Funciones · Web interactiva

Proyecto listo para GitHub Pages.

## Publicación
1. Sube **todo el contenido de esta carpeta** al repositorio.
2. En GitHub activa **Settings > Pages**.
3. Elige **Deploy from a branch**.
4. Selecciona **main** y **/(root)**.

## Estructura esperada
- `index.html`
- `pages/*.html`
- `css/styles.css`
- `js/ggb-helper.js`
- `assets/moto-mates.png`
- `.nojekyll`
