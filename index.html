Web interactiva del tema de funciones

Esta versión está preparada para GitHub Pages y organizada en páginas independientes. Cada bloque desarrolla una parte del tema con fórmulas escritas en LaTeX y una construcción dinámica de GeoGebra.

Idea clave: una función asigna a cada valor de \(x\) un único valor de \(y\).
