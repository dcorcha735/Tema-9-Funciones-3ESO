Inicio · 2. Dominio y recorrido

Dominio e imagen

El dominio es el conjunto de valores de x para los que la función existe. La imagen o recorrido es el conjunto de valores de y que alcanza.

En f(x) = 1/(x − a), el dominio es ℝ \ {a}.
En g(x) = √(x − b), el dominio cumple x ≥ b.
